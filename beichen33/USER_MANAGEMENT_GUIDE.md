# 用户管理功能使用指南

## 📍 访问路径

**管理员界面** → **系统配置** → **用户管理**

或直接访问：http://localhost:8892/system/users

---

## 🎯 功能说明

用户管理模块用于管理系统中的所有用户账号，包括：
- **管理员** (ADMIN)
- **教师** (TEACHER)  
- **家长** (PARENT)

---

## ✨ 核心功能

### 1. 查看用户列表
- 显示所有用户的详细信息
- 支持按角色筛选
- 实时显示用户状态（启用/禁用）

### 2. 添加用户
**步骤：**
1. 点击右上角 **"添加用户"** 按钮
2. 填写用户信息：
   - **姓名**（必填）
   - **邮箱**（必填，用作登录凭证）
   - **身份证号**（选填，也可用于登录）
   - **角色**（必填：管理员/教师/家长）
   - **所属校区**（必填）
   - **手机号**（选填）
   - **密码**（必填，默认：123456）
   - **状态**（启用/禁用）
3. 点击 **"确定"** 保存

**注意事项：**
- 邮箱必须唯一，不能重复
- 默认密码为 `123456`
- 教师账号会自动关联到现有教师数据

### 3. 编辑用户
**步骤：**
1. 在用户列表中找到要编辑的用户
2. 点击 **"编辑"** 按钮
3. 修改需要更新的信息
4. 点击 **"确定"** 保存

**注意事项：**
- 邮箱不可修改
- 密码留空则不修改
- 可以更改用户状态启用/禁用

### 4. 删除用户
**步骤：**
1. 在用户列表中找到要删除的用户
2. 点击 **"删除"** 按钮
3. 确认删除操作

**注意事项：**
- 删除为软删除，数据仍保留在数据库中
- 已删除的用户无法登录系统

---

## 📊 用户信息说明

| 字段 | 说明 |
|------|------|
| 姓名 | 用户真实姓名 |
| 邮箱 | 登录凭证（唯一） |
| 身份证号 | 可选登录凭证 |
| 角色 | 管理员/教师/家长 |
| 状态 | 启用/禁用 |
| 所属校区 | 用户所属的园区 |
| 手机号 | 联系电话 |
| 创建时间 | 账号创建日期 |

---

## 🔐 角色说明

### 管理员 (ADMIN)
- 访问管理后台桌面端
- 拥有所有权限
- 可以管理所有用户和数据

### 教师 (TEACHER)
- 登录后跳转到教师移动端
- 可以进行考勤、记录等日常操作
- 只能查看和管理自己的班级

### 家长 (PARENT)
- 登录后跳转到家长移动端（开发中）
- 只能查看自己孩子的信息
- 可以填写学校推送的表单

---

## 💡 使用技巧

1. **批量添加用户**
   - 可以多次点击"添加用户"快速创建多个账号

2. **关联现有教师**
   - 添加教师账号时，系统会自动关联到现有的教师数据
   - 邮箱应与教师信息中的邮箱保持一致

3. **重置密码**
   - 编辑用户时，填写新密码即可重置
   - 建议通知用户更改默认密码

4. **禁用账号**
   - 不想删除但需要暂时禁止登录时，将状态改为"禁用"

---

## ⚠️ 注意事项

1. **删除前请确认**
   - 删除用户会影响其关联的数据
   - 建议先禁用而非直接删除

2. **邮箱唯一性**
   - 同一邮箱不能创建多个账号
   - 修改邮箱需要删除后重建

3. **默认密码安全**
   - 提醒用户登录后修改默认密码
   - 增强账号安全性

## 🔧 API接口

如需程序化管理用户，可使用以下API：

```
GET    /api/users           # 获取用户列表
POST   /api/users           # 创建用户
GET    /api/users/:id       # 获取单个用户
PUT    /api/users/:id       # 更新用户
DELETE /api/users/:id       # 删除用户
```

**Swagger文档**: http://localhost:8891/api-docs
